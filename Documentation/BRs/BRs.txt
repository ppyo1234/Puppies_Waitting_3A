                                                                                     Puppies Waiting

Requisitos del negocio (BRs).

1.	Registro de usuarios:
El sistema tendrá un login para registrar usuarios con:
•	Nombre completo.
•	Correo electrónico.
•	Contraseña

2.	Registro de perritos en adopción.
El sistema tendrá un registro de perritos rescatados y que estarán a disposición del 
público para adopción o ayuda.
•	Foto
•	Nombre
•	Sexo
•	Raza (si aplica)
•	Condiciones 
•	Descripción adicional

3.	Recaudación de fondos y donaciones.
El sistema tendrá un formulario para recaudar fondos y o donaciones para ayudar a mantener
a los perros rescatados para alimentación y consultas veterinarias.

4.	Solicitudes de adopción 
El sistema contará con solicitudes de adopción para obtener datos de los posibles adoptantes.

5.	Seguimiento de adopción
Gracias a las solicitudes se podrán dar seguimiento a las adopciones para comprobar que los animales 
rescatados tengan una vida de calidad y mejor a la de antes de ser rescatados.

6.	Base de datos sobre veterinarias y refugios
El sistema dará informes y ubicaciones de veterinarias y refugios que brindaran apoyo a los usuarios.

7.	Blog de usuarios 
Abra un apartado donde el usuario podrá dejar recomendaciones y sus opiniones sobre la página 
y datos mostrados.
